Introduction

What is Kloudbit?

Kloudbit is a serverless platform designed to help developers build robust web and mobile applications without the hassle of back-end coding and server management.

Getting Started

1. Create an Application

Begin simply by creating an application.

Kloudbit applications are served by a RESTful API. To create your application, you will start by defining API resources that represent your business objects. Kloudbit will then automatically generate POST, GET, PUT and DELETE methods for your API resource.

All Kloudbit applications come with a User resource by default.
